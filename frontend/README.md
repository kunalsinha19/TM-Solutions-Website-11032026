# Frontend

Next.js marketing website for Tara Maa Solutions.

## Homepage sections

- Hero
- What we do
- Product categories
- About us
- Quote form
- Footer

## Run

```bash
npm install
npm run dev
```

## Environment variables

- `NEXT_PUBLIC_API_URL=http://localhost:5000/api`
- `NEXT_PUBLIC_RECAPTCHA_SITE_KEY=your-google-recaptcha-site-key`
