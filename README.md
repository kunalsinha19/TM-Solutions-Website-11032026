# Tara Maa Solutions

Monorepo scaffold for a B2B product website with:

- `apps/web`: Next.js + Tailwind CSS + Framer Motion
- `apps/api`: Node.js + Express + MongoDB
- `packages/*`: shared types, validation, config, and UI

## Next steps

1. Install dependencies with your preferred package manager.
2. Add environment variables for MongoDB, JWT, OTP, email, and captcha providers.
3. Start the API and web apps independently.
