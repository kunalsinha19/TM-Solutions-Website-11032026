# Admin

React admin dashboard for Tara Maa Solutions with:

- OTP login flow
- product add, edit, delete
- image upload and preview
- category management
- SEO page management
- quote request review
- website settings management

## Run

```bash
npm install
npm run dev
```

Set `VITE_API_URL` if the backend is not on `http://localhost:5000/api`.
