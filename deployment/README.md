# Tara Maa Solutions Deployment

## Quick start

1. Copy `backend/.env.example` to `backend/.env`
2. Copy `frontend/.env.example` to `frontend/.env.production`
3. Run `npm run install:all`
4. Run `npm run build:prod`
5. Start backend with `npm run start:backend`
6. Start frontend with `npm run start:frontend`

## GoDaddy production files

- `deployment/godaddy/README.md`
- `deployment/godaddy/ecosystem.config.cjs`
- `deployment/godaddy/nginx.conf`
