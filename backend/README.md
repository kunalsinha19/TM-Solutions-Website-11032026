# Backend API

Express + MongoDB backend for:

- Admin authentication
- JWT auth
- OTP email verification
- Backup email password recovery
- Product CRUD
- Category CRUD
- SEO page CRUD
- Quote request API
- Captcha validation
- SMTP email delivery with Nodemailer

## Environment variables

Create `.env` with:

- `PORT`
- `MONGODB_URI`
- `JWT_SECRET`
- `CLIENT_URL`
- `EMAIL_FROM`
- `ADMIN_NOTIFICATION_EMAIL`
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASS`
- `CAPTCHA_SECRET`
- `CAPTCHA_SITE_KEY`
